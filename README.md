## Clarify

Clarify is an AI-powered reading experience designed to help you actually understand what you're reading — not just scroll through it.

Built as a native iOS/macOS app, Clarify strips away distractions, enhances focus, and integrates an intelligent assistant directly into the reading flow so comprehension becomes effortless.

---

### ✨ Features

🧠 AI-Powered Understanding
- Ask questions directly about what you're reading
- Get summaries, key points, and explanations instantly
- Designed to improve comprehension, not replace thinking

📖 Focused Reading Modes
- **Center Stage**: Emphasizes the current reading line
- **Tunnel Vision**: Reduces peripheral distractions
- **Bionic Reading**: Guides your eyes for faster reading

🎨 Customizable Experience
- Font size, spacing, and style controls
- Light and dark mode support
- Clean, minimal UI designed for long reading sessions

---

### 🚀 Vision

Most reading tools optimize for speed or storage.

Clarify optimizes for **understanding**.

The goal is simple:
> Help users read faster, retain more, and feel confident in what they learn.

---

### 🛠️ Tech Stack

- **Frontend:** Swift (iOS/macOS)
- **Backend:** Supabase (PostgreSQL + Auth + API)
- **Architecture:** Client-side reading engine with backend-supported data storage
- **AI Integration:** Context-aware assistant embedded into reading flow
